
# DIVAPOS SERVICE

Aplikasi kasir penjualan dan Servis.
Software ini bisa digunakan untuk melakukan transaksi penjualan serta pengelolaan servis.
Cocok untuk Anda yang memiliki usaha servis komputer, handphone dan elektronik.

## Features

- Dashboard
- Master Barang
- Master Pelanggan
- Master Supplier
- Point Of Sale (Kasir)
- Terima Servis
- Ambil Servis
- Laporan Data Servis
- Laporan Penjualan
- Laporan Pembelian
- Laporan Laba Rugi
- Return Penjualan
- Return Pembelian
- Grafik Penjualan
- Laporan Cashflow
